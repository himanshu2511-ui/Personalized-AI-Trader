# Personalized AI Trader

Personalized AI Trader is a comprehensive intelligence trading platform that empowers investors to make data-driven decisions using AI-powered analysis. Built entirely on the client side, it simulates a real-world algorithmic trading environment to help users reduce emotional bias, manage risk effectively, and optimize their trading strategies through clear visualizations and actionable recommendations.

The platform acts as a Decision Support System that analyzes market data, suggests optimal entry/exit points, applies automated risk controls, and provides detailed performance analytics — all without requiring backend servers or real money.

## Key Features

* **AI-Driven Market Intelligence:** Analyzes selected stocks or assets using algorithmic models to generate intelligent insights, trend predictions, and trading signals.
* **Data-Backed Decision Support:** Delivers clear entry and exit recommendations backed by technical indicators (SMA, RSI) and AI logic, helping users trade with confidence.
* **Advanced Risk Mitigation:** Implements smart stop-loss and target prices to prevent catastrophic losses and lock in profits.
* **Performance Analytics Dashboard:** Tracks key metrics in real-time, including:
  * Win rate
  * Overall strategy performance / PnL
* **Financial Data Visualization:** Beautiful, interactive candlestick charts built with TradingView Lightweight Charts for intuitive understanding of market movements.
* **Futuristic UI/UX:** A responsive, modern interface with a glassmorphism design system focused on usability and aesthetics for traders.

## Technology Stack

* **Core:** Vanilla JavaScript (Client-side only)
* **Build Tool:** Vite
* **Visualization:** TradingView Lightweight Charts
* **Icons:** Lucide
* **Styling:** Custom Vanilla CSS (Glassmorphism, Dark/Neon mode)

## How It Works (User Flow)

1. **Select Assets** — Choose symbols from the sidebar (BTC, ETH, AAPL, TSLA, NVDA) you want to analyze.
2. **AI Analysis** — The platform instantly processes simulated real-time intraday data using built-in technical models.
3. **Get Recommendations** — View AI-suggested signals (Buy/Sell/Hold), calculated targets, stop-loss, and confidence scoring.
4. **Simulate Trades** — Execute paper trades with your virtual $100,000 balance and watch the strategy perform.
5. **Review Analytics** — Dive into the Performance table to track active positions and review closed trade P&L.

## Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/himanshu2511-ui/Personalized-AI-Trader.git
cd Personalized-AI-Trader
```
2. Install dependencies:
```bash
npm install
```
3. Run the development server:
```bash
npm run dev
```
4. Open your browser and navigate to the local host address provided (typically `http://localhost:5173`).
5. Visit:  https://personalized-ai-trader.vercel.app/ for experiencing the AI signals and make trades.
